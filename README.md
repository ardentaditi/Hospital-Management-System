# Hospital Management System (Spring Boot)

A backend web application developed using Java Spring Boot to manage hospital operations such as patients, doctors, and appointments.

## Features
- Patient registration and management
- Doctor management
- Appointment booking
- RESTful APIs
- Layered architecture (Controller, Service, Repository)
- Database integration using Spring Data JPA

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL / H2
- Maven

## Project Structure
- Controller Layer – Handles HTTP requests
- Service Layer – Business logic
- Repository Layer – Database access
- Entity Layer – Data models

## How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Configure database in `application.properties`
4. Run the main Spring Boot application
